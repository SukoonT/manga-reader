keyboard shortcuts:-

| Keys                  | Function                        |
| --------------------- | ------------------------------- |
| "wsad and arrow keys" | scroll                          |
| " - and = and + "     | size                            |
| "spacebar"            | big scroll                      |
| "h"                   | home                            |
| "f"                   | go to page                      |
| "b"                   | bookmark                        |
| "f1"                  | github page                     |
| "< and >"             | next/prev                       |
| "Enter or /"          | focus input box(home page only) |

move mouse to left edge for list while in reader.
